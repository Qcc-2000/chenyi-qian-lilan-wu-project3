### API Docs

```bash
cd backend

```

### Run

1. Install dependencies

   ```bash
   cd backend
   npm install
   ```

2. replace the `.env.example` file with `.env` and set the environment variables

3. Run the backend

   ```bash
   npm run dev
   ```
